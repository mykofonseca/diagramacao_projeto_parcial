Introdução ao projeto: Help! SENAI

Introdução: Website para Instituição Escolar - Uma Ponte Digital entre a Escola e o Aluno

Requisitos funcionais

Nº Requisito Descrição
1 O sistema deve permitir que o usuário efetue login com suas credenciais fornecidas pela instituição.
2 O sistema deve verificar as credenciais do usuário com o sistema de autenticação da instituição
3 O sistema deve armazenar as informações do usuário após o login bem-sucedido.
4 O sistema deve exibir uma página principal com uma amostra de mapas das áreas da instituição.
5 A página principal deve ser intuitiva e fácil de navegar.
6 O sistema deve permitir que a instituição escolar adicione, edite e remova eventos.
7 O sistema deve permitir que os usuários pesquisem por eventos por data, categoria, nome ou palavra-chave
8 O sistema deve exibir uma lista de eventos com datas, categorias e descrições.
9 O sistema deve fornecer uma interface de chat em tempo real entre usuários e a instituição escolar.
10 O sistema deve permitir que os usuários enviem mensagens e arquivos.
11 O sistema deve armazenar o histórico das conversas.
12 O sistema deve permitir que a instituição escolar adicione, edite e remova datas importantes no calendário.
13 O sistema deve exibir o calendário em diferentes visualizações (mensal, semanal, diária).
14 O sistema deve permitir que os usuários visualizem as datas importantes.
15 O sistema deve permitir que os usuários alterem sua senha, email ou número de telefone.
16 O sistema deve solicitar a autenticação do usuário antes de permitir a modificação de informações.
